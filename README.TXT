Инструкция к применению бота:

1. Нажмите кнопку START
2. Появятся 2 кнопки: Погода и Анекдот.
3. Если хотите получить рандомный анекдот, то просто нажмите соответствующую кнопку.
4. Если хотите получить сведения о погоде на данный момент + прогноз на день и на неделю, то жмите кнопку "Погода".

!Внимание! Для прогноза погоды требуется знать местоположение запросившего, поэтому клиент телеграм запросит подтверждение на отправку геолокации.
На данный момент такая функция поддерживается только в мобильном клиенте. Поддержки desktop-версии на данный момент нет.
